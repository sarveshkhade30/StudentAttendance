# Student Attendance Management System

Java + MySQL (JDBC) console app to add students and mark attendance.

## Setup
1. Install Java JDK 17+ and MySQL.
2. Run `db/schema.sql` in MySQL Workbench to create `attendance_db` and sample data.
3. Put `mysql-connector-j-<version>.jar` in project root.
4. Edit password in `src/DBConnection.java` if needed.
